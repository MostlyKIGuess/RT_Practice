# RT_Practice
Ray tracing

## How to Run

### Prerequisites
- Ensure you have SDL2 installed on your system. You can install it using the following command:
  ```sh
  sudo apt-get install libsdl2-dev
  ```
### Compilation
```
g++ -o pipes main.cpp -lSDL2
```
### Running
```
./pipes
```
or
```
./lighted > image.ppm
```
